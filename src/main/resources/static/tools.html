RAG Evaluation API Tools

Explore and test the RAG evaluation endpoints

GET

Retrieve Documents

GET /getDocuments?documentText={query}

Parameters:

documentText

The query text to find similar documents

Returns a list of relevant documents
GET

Evaluate RAG Performance

GET /evalRag?documentText={query}

Parameters:

documentText

The query text to evaluate RAG performance

{ "precision": 0.2, "recall": 0.011, "f1Score": 0.021, "mrr": 0.0, "ndcg": 0.0, "hitRate": 0.0 }
POST

Store Document

POST /storeDocument?documentText={text}

Parameters:

documentText

The document text to store in the RAG system

"Document stored successfully"

Testing the API

You can test these endpoints using any HTTP client. Here's an example using curl:

# Store a document
curl -X POST "http://localhost:7860/storeDocument?documentText=Example%20document"

# Retrieve similar documents
curl "http://localhost:7860/getDocuments?documentText=dishwasher"

# Evaluate RAG performance
curl "http://localhost:7860/evalRag?documentText=dishwasher"
